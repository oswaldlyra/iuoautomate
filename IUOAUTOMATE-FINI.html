Guide d'utilisation de l'IUOAUTOMATE-FINI

Activer le bouton "Créer Automate" pour créer les états et les transitions
- Pour créer un état il suffit de faire un clic droit dans la zone blanche(Canevas). Puis cliquer sur créer nouveau état pour saisir le nom souhaité à cet état. Il faut répéter l’action pour créer tous les états de l’automate à créer.

- Pour créer une transition, il faut glisser depuis l’état source jusqu’à l’état de destination. Un message vous demandera de saisir le symbole de la transition. Il faut répéter l’action pour créer tous les états de l’automate à construire.

- Une fois l'état est créé, clic droit sur l'état, puis indiquer si l'état est final ou initial

Après la création de l’automate, vous pouvez effectuer des opérations.

Activer le bouton "Bouger Etat", puis cliquer/glisser sur un état pour le déplacer.
- Cliquer sur "Minimiser AFD" pour minimiser l'Automate Fini créé.
- Cliquer sur "Langage Régulier" pour trouver le code de l'Automate Fini créé.
- Cliquer sur "Vérifier Chaine" pour vérifier si un mot appartient à l'Automate Fini créé.
- Cliquer sur "Enregistrer Image" pour Capturer le Canevas.
- Cliquer sur "Nettoyer Canevas" pour effacer l'Automate créé.

Note :
Un seul état peut-être initial. Si vous avez choisi un état initial par erreur, vous pouvez le changer en faisant un clic droit sur l'état souhaité
Quand vous effectuez un clic droit sur un état, Une fénêtre apparaitra. Vous pouvez renommer et supprimer l'état, renommer et supprimerune transition.
